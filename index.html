Introducción al lenguaje HTML

Introducción - Lenguaje de marcas

HTML es un lenguaje de marcas, es decir, es una forma de definir texto con etiquetas. Estas etiquetas tienen (generalmente) un nombre y un contenido, y se define <etiqueta>contenido</nombre>.

Las etiquetas se tienen que declarar en una jerarquía, y en las versiones anteriores a HTML5 la única etiqueta que podía existir en el primer nivel era la etiqueta html, la cual solo contiene las etiquetas head (información sobre el documento) y body (contenido del documento).

HTML ignora los espacios en blanco redundates, así que podemos distribuir una línea en múltiples dentro del código sin que afecte a su visualización.

Entidades

Las entidades son formas de representar carácteres no válidos o no reconocidos en el charset del documento. Con ello, podemos representar carácteres especiales, o código, por ejemplo, podemos representar el código <p>Hola Mundo</p> escribiendo lo siguiente en el documento HTML: &lt;p&gt;Hola Mundo&lt;/p&gt;

Etiquetas

Enlaces

La etiqueta <a> (abreviación de "anchor") representa un enlace a otro documento o contenido. Estas etiquetas tienen que tener como mínimo la URL o la tag de la página a la que dirigir al usuario.

Cabeceras / títulos

Las etiquetas <h1>, <h2> ... <h6> representan los encabezados (la "h" es una abreviación de header). Un ejemplo de la aplicación de esta etiqueta se encuentra en los encabezados de cada sección de este documento.

Etiquetas de estilo

Son etiquetas cuyo uso no se suele recomendar en HTML5, estas etiquetas nos permite aplicar estilos sobre el texto como destacarlo (etiqueta "strong") o hacer énfasis (etiqueta "em").

Etiquetas de contextualización de texto

Son etiquetas cuya finalidad es marcar un texto que debe ser interpretado de una forma particular por los navegadores. Un ejemplo de ello pueden ser las citas (etiqueta "q"), o el código (etiqueta "code").

Enlaces

Los documentos HTML tienen muchos enlaces que se cargan automáticamente por los navegadores. Estos elementos son frecuentemente metadata (etiquetas "meta"), hojas de estilo (incluídas en las etiquetas "link") o código JavaScript (etiquetas "script").

En este documento no se incluirá ejemplo de estas etiquetas porque generalmente deben ir en el header, y también porque en esta práctica se nos especifica no implementar este tipo de etiquetas.

Listas

Los documentos HTML pueden incluir listas numeradas y no numeradas, las cuales se definen respecticamente con los elementos "ul" (abreviación de "unordered list") y "ol" (abreviación de "ordered list"), y sus elementos se definen con la etiqueta "li" (abreviación de "list item").
